# OnecaratEditor — Portable Edition

> Version: See `VERSION.txt`

## Getting Started

1. Extract all files from the ZIP archive.
2. Keep the folder structure as-is.
3. Run `OnecaratEditor.exe`.

## Notes

- This package is the **portable edition** — no installation required.
- Do not move or delete files in the `binaries` or `resources` folders.
- Some features depend on bundled local binaries.
- Do not run the executable directly from the ZIP file.

## Security

- This application runs **entirely locally**.
- No automatic external communication occurs by default.

## First Launch on Windows

When running OnecaratEditor for the first time, Windows SmartScreen may display a warning because the application is new.

If this happens:

1. Click **"More info"**
2. Click **"Run anyway"**

This is expected behavior for newly released applications without an extended validation certificate.
